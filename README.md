# 핏코치 (FitCoach) 🔥

188cm · 113kg → **목표 83kg (−30kg)** 개인 맞춤 다이어트·근력·유산소·식단 코칭 웹앱.
운동 초보 · 무릎 보호 우선 · **주4회 4분할(가슴삼두·등이두·하체·팔어깨)** · **계란 제외** 한식 식단에 맞춰 설계되었습니다.

> 경쟁 앱 12개+ 리서치 → 블루프린트(`docs/BLUEPRINT.md`) → MVP 구현 순서로 만들어졌습니다.

## 실행 방법

### ✅ 1) 가장 쉬움 — `핏코치.html` 더블클릭 (권장)
**`핏코치.html` 파일을 더블클릭**하면 끝입니다. React·Tailwind·앱 코드가 모두 내장되어 **인터넷 연결 없이도** 동작합니다. (한글 폰트만 인터넷이 있으면 더 예쁘게 보임)
> 실행이 안 되던 분은 이 파일을 여세요. `index.html`을 더블클릭하면 브라우저 보안(CORS) 때문에 빈 화면이 뜹니다 — 그건 아래 2)번 방식으로 열어야 합니다.

### 2) 로컬 서버로 `index.html` 열기 (개발용)
`App.jsx`를 분리해서 쓰는 방식. `file://` 더블클릭은 안 되고 **로컬 서버**가 필요합니다.
```bash
# 폴더에서 아래 중 하나 실행 후 안내된 주소 접속
npx serve            # → http://localhost:3000
# 또는
python -m http.server 8000   # → http://localhost:8000
```
VS Code라면 **Live Server** 확장으로 `index.html` 우클릭 → "Open with Live Server"도 가능.

### 3) Vite/React 프로젝트에 넣기 (배포·확장용)
`App.jsx` 상·하단을 아래처럼 바꾸면 표준 모듈이 됩니다.
- 맨 위: `import React, { useState, useEffect, useRef, useMemo } from 'react'`
- 맨 위 `const { useState, ... } = React;` 줄 삭제
- 맨 아래 `const root = ReactDOM...` / `root.render(...)` 두 줄 삭제 → `export default App`
- Tailwind를 프로젝트에 설정(또는 `index.html`에 Tailwind CDN 유지)

## 주요 기능

| 영역 | 기능 |
|---|---|
| 🏠 홈 | 목표 진척률, 오늘의 분할+바로 시작, 이번주 4회 링, 목표 칼로리/매크로, 체중 추세 미니그래프, 1분 레슨, 다음 마일스톤 |
| 💪 운동(근력) | 3단계(적응→발달→강화) × 4분할, 운동별 **근육맵·자세 큐·유튜브 시연**, **세트 로깅(직전기록 자동채움)**, **휴식 타이머 자동시작**, **규칙형 다음 무게 제안 + RiR(여유)**, **워밍업 세트 자동생성**, **플레이트 계산기**, **무릎 위험 플래그 + 대체운동 1탭** |
| 🏃 운동(유산소) | 저충격 종목 가이드, 개인 심박존, 단계별 계획, 걸음수(NEAT) 목표, 유산소 타이머 |
| 🍱 식단 | **신호등 색(녹/노/빨) + 숫자**, **계란 자동 제외**, 단백질 최우선 게이지, 음식 검색·기록, 추천 3일 식단, 음식 DB, 외식/편의점 꿀팁, 물 추적 |
| 📈 진행 | **7일 이동평균 추세선** + 목표선 + 예상 도달주, 습관 체크리스트, 감량 마일스톤, 운동 기록·총 볼륨, 정체기/타임라인 코칭, 교육 레슨, **JSON 백업 내보내기** |
| 온보딩 | 목표→무릎 체크→칼로리 처방 + 의료 면책 |

칼로리/단백질은 **Mifflin-St Jeor 공식**으로 신체정보·현재 체중에 맞춰 자동 계산됩니다. (헤더 "내 정보"에서 수정)

## 파일 구성
```
fitness-coach-app/
├─ 핏코치.html       # ✅ 더블클릭 실행 (React·Tailwind·앱 전부 내장, 오프라인 OK)
├─ App.jsx           # 앱 소스 (단일 파일, React) — 핏코치.html은 이걸 빌드한 결과
├─ index.html        # App.jsx 를 불러와 실행 (로컬 서버 필요, 개발용)
└─ docs/
   ├─ BLUEPRINT.md   # 제품 블루프린트(기능 매트릭스·IA·로드맵)
   └─ research_1~6.md# 경쟁 앱 리서치 원자료
```
> `핏코치.html`을 다시 빌드하려면: `App.jsx`를 esbuild로 컴파일하고 Tailwind CSS를 생성해 React UMD와 함께 인라인합니다. (수정은 `App.jsx`에서 하세요.)

## 데이터 & 프라이버시
- 모든 기록(체중·운동·식단·습관)은 **브라우저 localStorage**에만 저장됩니다(서버 전송 없음).
- 브라우저 캐시를 지우면 데이터가 사라지니, **진행 탭 → "내 데이터 백업"**으로 JSON을 주기적으로 내보내세요.

## ⚕ 면책
본 앱은 의료 조언이 아닌 일반 건강·운동 정보를 제공합니다. 고체중·기저질환이 있다면 운동 시작 전 의사와 상담하고, 통증·이상 증상 시 즉시 중단하세요. 운동 시연 영상은 유튜브 검색으로 연결되며 앱은 영상을 재호스팅하지 않습니다.
