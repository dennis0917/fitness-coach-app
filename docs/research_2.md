# 근력운동 기록/트래커 앱 리서치 (2024~2026 최신 기능 기반)

> 우리 앱 관점: 188cm·113→83kg·운동 초보·무릎 보호 필요. 평가 축은 ①휴식 타이머 ②점진적 과부하/다음 무게·횟수 제안 ③루틴/분할 ④세트 로깅 UX ⑤플레이트 계산기 ⑥기록 그래프 + 초보 가르치기(시각자료).

---

## 1. Hevy
- **카테고리/플랫폼**: 근력 기록 + 가벼운 소셜, iOS/Android/Web/Apple Watch·Wear OS. 프리미엄.
- **핵심 기능**
  - **휴식 타이머(우리 핵심)**: 루틴 설계 시 운동별로 5초~5분 개별 설정. 세트를 '완료' 체크하면 **자동 카운트다운 시작**, 종료 시 알림. 기본 타이머 off 가능(설정 > Workouts > Default Rest Timer). iOS **Live Activity 위젯**으로 폰 잠금/타 앱 사용 중에도 타이머·현재 운동·다음 세트·목표 횟수 표시.
  - **점진적 과부하/다음 무게 제안**: 핵심은 **직전 기록 자동 표시(이전 무게·횟수를 입력란에 회색으로 미리 채움)** — "지난번에 뭘 했는지" 즉시 보여주는 방식. 유료 **Hevy Trainer(어댑티브 프로그래밍)**는 점진적 과부하 내장 프로그램을 생성하고 성과 추적 후 작업 무게를 자동 조정. 무게 목표 기반 **워밍업 세트 계산기**(Pro)도 있음.
  - **루틴/분할**: 커스텀 루틴, 운동별 세트수·목표 횟수 범위·자동 휴식시간 지정. 25개 이상 프리셋 프로그램 라이브러리. 400개 이상 운동(약 100개 이상은 단계별 설명 영상 포함).
  - **세트 로깅 UX**: 빠른 체크 방식, 슈퍼세트/드롭세트 지원. 무게/횟수 자동 채움으로 입력 최소화.
  - **플레이트 계산기**: 내장. 바벨/숏바/EZ바에서 보유 플레이트 기준 한쪽에 끼울 플레이트 조합 계산.
  - **그래프**: PR, 볼륨, 1RM 추정 등 통계/그래프 제공.
- **우리 앱 적용 점(takeaway)**: ①"이전 기록을 입력란에 회색 placeholder로 미리 채우는" 패턴은 점진적 과부하 UX의 최소비용·최대효과 사례 → 우리도 기본 채택. ②세트 완료 체크 → 휴식 타이머 자동 시작 → 알림 흐름을 그대로 따르되, 웹앱이므로 백그라운드/잠금 알림은 한계가 있으니 화면 내 큰 카운트다운 + 소리/진동으로 보완. ③초보용으로 운동별 설명 영상/이미지 동봉 방향이 맞음.
- **강점**: 무료 한도가 비교적 관대(루틴 4개·커스텀 운동 7개·기록 3개월). 가격 저렴(Pro 월 $2.99 수준 언급). 깔끔한 UX + 영상 라이브러리.
- **약점/주의**: 자동 무게 제안의 '지능'은 무료에선 약함(직전 기록 표시 위주), 진짜 AI 조정은 유료 Trainer. 무료는 데이터 3개월/루틴 4개 제한.
- **수익모델**: 프리미엄(Free vs Pro). Pro 저가, Hevy Trainer/Coach 별도.

## 2. Strong
- **카테고리/플랫폼**: 미니멀 근력 로거, iOS/Android/Apple Watch. 프리미엄.
- **핵심 기능**
  - **휴식 타이머**: 세트 입력 후 설정된 휴식시간으로 **자동 카운트다운 시작**, 운동별 개별 휴식시간 지정 가능.
  - **점진적 과부하/다음 무게 제안**: **직전 워킹세트 기준 1RM 자동 추정**(저반복은 Brzycki, 고반복은 Epley 공식; RPE 입력 시 정확도 향상). 운동별 추정 1RM 추이를 시간에 따라 표시. 즉, "AI 추천"보다는 **이전 기록 + 1RM 추정치 제공** 방식.
  - **루틴/분할**: 커스텀 루틴 템플릿. 무료는 **루틴 3개 제한**(워크아웃 로깅 자체는 무제한).
  - **세트 로깅 UX**: 업계에서 가장 빠르고 깔끔한 로깅으로 정평. 무료에도 450+ 운동·기록·1RM 계산기 포함.
  - **플레이트 계산기**: 모든 워킹세트에 내장 — 목표 무게 입력 + 보유 플레이트 선택 시 한쪽 바에 끼울 조합 안내.
  - **그래프**: 무료에 전체 기록, 유료에 볼륨/빈도 분석·1RM 추이·CSV 내보내기.
- **우리 앱 적용 점**: ①플레이트 계산기를 "모든 워킹세트 안에 자연스럽게 내장"하는 방식 참고(고체중 초보가 무릎 보호용으로 정확한 바벨 세팅 필요). ②1RM 공식 두 가지(저/고반복 분기)를 다음 무게 제안 로직 근거로 사용 가능 — 우리도 Epley/Brzycki로 추정 1RM 계산 후 목표 %로 다음 무게 역산.
- **강점**: 로깅 속도/단순함 최고 수준, 입문자 진입장벽 낮음.
- **약점/주의**: 무료 루틴 3개 제한이 4분할(우리 요구)엔 빠듯. AI 자동조정 없음(사용자가 직접 무게 올림). 가격 정보 출처마다 상이(월 $9.99/연 $49.99 ~ 일부 $4.99/$29.99) → [추정] 변동 가능.
- **수익모델**: 프리미엄.

## 3. Fitbod
- **카테고리/플랫폼**: AI 자동 프로그래밍의 대표주자, iOS/Android/Apple Watch. 구독 전용(무료 체험만).
- **핵심 기능**
  - **다음 무게·횟수 제안(우리 핵심, 가장 정교)**: 알고리즘이 ①로그 기반 **추정 1RM** 계산 후 매 세션 갱신 ②**근육별 회복도(%)**(80~100% = fresh) ③**훈련 볼륨 추세**(세트×횟수×무게; 주당 ~10–15% 증가가 최적이라는 자체 데이터) ④세트별 **RiR(Reps in Reserve, 남은 반복수)** 피드백 ⑤장비/직전 세션 시점 등을 종합해 **다음 세션의 무게/횟수를 자동 제시**. 회복 부족·피로 신호 시 강도 낮추거나 다른 근육으로 전환.
  - **휴식 타이머**: 내장. 운동 유형에 맞는 적정 휴식시간으로 **자동 팝업**, 다음 세트 시점 알림.
  - **워밍업 세트**: 워킹 무게 기준 **워밍업 세트 자동 추가**(점진 가중). 슈퍼세트/드롭세트/AMRAP도 무리 없이 로깅.
  - **루틴/분할**: 사용자가 분할을 짜기보다 **AI가 매일 운동을 생성**(장비·회복·목표 반영). 직접 구성도 가능.
  - **운동 시각자료(초보 가르치기)**: 1,000+ 운동, **고해상도 다각도 영상** + 이해하기 쉬운 상세 지침.
  - **그래프/플레이트**: 진행 추적·볼륨 그래프 제공. (플레이트 계산기 내장 여부는 검색으로 명확히 확인 못함 — [추정] 워밍업/세트 화면에서 무게 가이드는 제공하나 별도 플레이트 계산 UI는 불명확.)
- **우리 앱 적용 점**: ①**RiR 입력 → 다음 세션 무게 조정** 루프가 우리 "다음 무게/횟수 자동 제안(#5)"의 가장 좋은 모델. 초보에겐 "몇 개 더 할 수 있었나?(0~4+)" 형태로 단순화해 받자. ②**근육 회복도 개념**으로 4분할 스케줄과 결합(같은 부위 연속 자극 방지 → 초보·고체중 부상 예방에 부합). ③**워밍업 세트 자동 생성**은 113kg 초보의 무릎/관절 보호에 직접 유용 → 채택. ④다각도 영상 = 우리 "운동별 시각자료(#6)" 벤치마크.
- **강점**: "다음에 얼마 들지" 자동 안내가 가장 강력하고 초보 친화적(생각 없이 따라가면 됨). 깔끔한 UX.
- **약점/주의**: **가장 비쌈** — 2026년 월 $15.99 / 연 $95.99(인상됨, 7일 무료체험). Apple Watch도 폰 근접 필요. 초보가 "왜 이 무게인지" 학습은 약함(블랙박스). 우리는 비상업/자체 앱이므로 이 알고리즘 아이디어만 차용하면 됨.
- **수익모델**: 구독 전용(고가).

## 4. Jefit
- **카테고리/플랫폼**: 대형 운동 DB + 커뮤니티 + 프로그래밍, iOS/Android. 프리미엄(광고형 무료).
- **핵심 기능**
  - **운동 DB/시각자료(강점)**: **1,400~1,500+ 운동**, **애니메이션 시연 + 근육 활성 다이어그램**, 부위/패턴/장비/난이도 필터. 초보용 그림 큐 + 설명. 2025 신규 **BodyMap**(근육 탭하면 해당 부위 기여 운동을 % 순위로 표시), **폼 점검용 영상 업로드**.
  - **점진적 과부하**: 2025 **AI 기반 점진적 과부하 시스템**(볼륨·강도·근력 추세 분석해 부하 진행 지원). 1RM 추정·고급 1RM 분석·근력 추적. 세트별 **인터벌 타이머**(목표 시간 카운트다운 + 기록).
  - **루틴/분할**: 850+ 짜여진 플랜 + 커뮤니티 루틴 다운로드 + 직접 설계. 분할 구성 자유도 높음.
  - **그래프**: 진행 분석/차트 다양.
- **우리 앱 적용 점**: ①**근육 다이어그램 + 운동별 기여도 시각화(BodyMap)**는 우리 "운동별 시각자료(#6, 자세/근육)" 요구에 정확히 부합 — 4분할 각 부위별로 어떤 근육을 때리는지 보여주는 화면 벤치마크. ②애니메이션 시연이 정적 이미지보다 초보 폼 학습에 유리.
- **강점**: 운동 라이브러리·시각자료·프로그램 양이 압도적. 초보가 "이 운동 어떻게 하지"를 해결하기 좋음.
- **약점/주의**: **무료 버전 광고가 운동 흐름을 끊을 만큼 거슬림**(잦은 불만). 기능/플랜이 너무 많아 초보가 **무엇을 골라야 할지 길잡이가 부족**(선택 과부하). UX 평가 보통(3/5 언급). 가격: Elite 월 $12.99 / 연 $69.99(과거 대비 인상).
- **수익모델**: 광고형 무료 + Elite 구독.

## 5. FitNotes
- **카테고리/플랫폼**: 초미니멀 무료 로거, **Android 전용**. 완전 무료·광고 없음·오프라인·계정 불필요.
- **핵심 기능**
  - **로깅 UX**: 속도/단순함 최우선. 즉흥 기록 또는 사전 정의 루틴 둘 다 지원. **직전 워크아웃 세트/횟수 복사** 기능으로 점진적 과부하 추적이 빠름.
  - **휴식 타이머**: 내장 휴식 타이머 있음. ([추정] 자동 시작/백그라운드 알림의 정교함은 Hevy/Strong보다 단순한 수동·기본형으로 알려짐 — 검색으로 자동시작 여부 확정 못함.)
  - **플레이트 계산기 + 그래프**: 모두 **무료·광고 없이** 제공.
  - **운동 DB**: 기본 운동 프리셋 + 커스텀 운동 추가.
- **우리 앱 적용 점**: ①"**직전 워크아웃 복사 → 약간 올려서 진행**"이라는 가장 단순한 점진적 과부하 모델 — AI가 과할 때의 미니멀 대안으로 참고(우리 앱에 '지난 기록 그대로 불러오기' 버튼). ②오프라인·계정불필요·무광고 = 비상업 자체 앱 철학과 부합.
- **강점**: 완전 무료·무광고, 빠르고 안정적, 군더더기 없음. 파워리프팅/순수 근력 로깅엔 최고의 무료 앱이라는 평.
- **약점/주의**: **iOS 미지원(Android 전용)**. 시각자료/운동 영상·소셜·AI 추천 없음 → 초보 "가르치기" 기능은 사실상 부재(우리 요구 ①⑥엔 부족). 자동 무게 제안 없음.
- **수익모델**: 없음(완전 무료, 기부형).

## 6. Liftin'
- **카테고리/플랫폼**: 미니멀 근력 로거, **iOS/iPad/Mac/Apple Watch 중심**(Apple 생태계). 프리미엄.
- **핵심 기능**
  - **다음 무게 제안**: **자동 무게 조정 규칙(automatic weight adjustment rules)** — 결과(달성 여부)에 따라 무게를 규칙 기반으로 자동 조정. (Fitbod식 AI보다 단순한 규칙형: 예 "목표 횟수 달성 시 다음에 +X" 형태로 추정.)
  - **그래프**: 단·장기 진행 시각화 그래프 강조.
  - **Apple Watch**: 손목에서 독립/연동 풀 워크아웃 추적, Apple Health 연동(활동링).
  - 운동 변수 거의 전부 커스터마이즈 가능하면서도 단순함 유지.
- **우리 앱 적용 점**: ①**"규칙 기반 자동 무게 조정"**은 AI 없이 구현 가능한 점진적 과부하의 현실적 모델 — 우리가 가볍게 시작하기 좋은 방식(목표 횟수 전부 달성 → 다음 세션 무게 +2.5/5kg 제안 등 결정론적 규칙). 초보에게 "왜 올랐는지" 설명도 쉬움(블랙박스 아님).
- **강점**: 단순함 + 자동 조정 규칙 + Apple Watch 통합. 라이프팅에만 집중해 잡기능 없음.
- **약점/주의**: **Apple 전용**, 무료는 **월 5회 워크아웃 제한**, 유료 연 $24.99. 시각자료/근육 다이어그램·초보 교육 콘텐츠는 약함. 인지도 낮음(검증 자료 적음, 일부 [추정]).
- **수익모델**: 프리미엄(연 구독).

- **Fitbod**: 추정1RM + 근육 회복도 + 볼륨추세 + RiR을 종합한 **AI 자동 제안**(가장 정교, 가장 비쌈, 블랙박스).
- **Liftin'**: 결과 기반 **규칙형 자동 조정**(결정론적, 설명 쉬움, AI 없이 구현 가능).
- **Hevy / Strong**: **직전 기록 자동 표시 + 1RM 추정**(사용자가 직접 올림). Hevy Trainer(유료)만 자동 조정.
- **FitNotes**: **직전 기록 복사** 후 수동으로 올림(가장 단순).
- **Jefit**: 2025 **AI 점진적 과부하 시스템**(추세 분석 기반 부하 진행) + 1RM 분석.

## 이 카테고리 핵심 교훈 3가지

1. **"이전 기록 자동 표시 + 다음 목표 자동 제안"이 점진적 과부하 UX의 핵심.** 가장 비용 대비 효과 큰 패턴은 Hevy/FitNotes식 **직전 세트를 입력란에 미리 채우는 것**이고, 여기에 우리는 **Liftin'식 규칙형 자동 조정**(목표 횟수 전부 달성 → 다음에 +2.5~5kg, 미달 → 유지/감량)을 얹는 게 현실적. Fitbod의 **RiR 피드백 루프 + 추정1RM(Epley/Brzycki)**는 아이디어만 차용해 초보용으로 단순화("몇 개 더 가능했나?"). 고가 AI를 그대로 흉내 낼 필요 없음.

2. **휴식 타이머는 "세트 완료 체크 → 자동 시작 → 알림"이 표준이며, 운동별 개별 시간 설정이 필수.** Hevy/Strong/Fitbod 모두 동일 흐름. 단, 우리는 **웹앱**이라 백그라운드/잠금화면 알림이 제한되므로 **화면 내 대형 카운트다운 + 소리/진동**으로 보완하고, 4분할 운동마다 기본 휴식시간(복합운동 길게/고립운동 짧게)을 프리셋으로 제공해야 함.

3. **초보·고체중·무릎보호 사용자에겐 "교육 + 안전"이 차별점이다.** 순수 로거(FitNotes/Strong)는 빠르지만 가르쳐주지 않음. 우리는 Jefit의 **근육 다이어그램/BodyMap + 애니메이션 시연**과 Fitbod의 **워밍업 세트 자동 생성**을 결합해, 운동별 자세·타깃 근육 시각자료와 점진적 워밍업으로 부상 위험을 낮춰야 함. 플레이트 계산기(Strong식 세트 내장형)도 정확한 바벨 세팅을 위해 기본 탑재. Jefit의 반례(광고·선택 과부하)를 보면, **단일 4분할 루틴으로 선택지를 좁혀 길잡이를 주는 것**이 초보에게 오히려 유리.
